TOFUSCRIPT ‚Äî The actually intuitive programming language

I created this programming language intending to let everyone, even those without programming experience, write meaningful code. To do this, I went back to a principle of UX design, mold the product after your user's view of the world, not create a product a force the users to learn it. Thus, I created a programming language where functions look like algebraic functions, and objects are created from a blueprint. I am motivated to create this language because I remembered the struggles of learning how to code without a professor at my side (unfortunately, a professor isn't a luxury everyone has). I want to be able to help teach programming and the first step I can take is to create a language that is easy to teach and easy it learn. I specifically based the syntax design base off of how humans already interact with the logical world. For more about this project, visit the GitHub repo at https://github.com/bfkwong/TofuJS

LATEX BOOK PARSER ‚Äî Conversion from Markdown or iPynb to Overleaf Project

This project came from a frustration of mine that I cannot convert my Jupyter Notebooks to an overleaf project correctly. While NBConvert does some things right, it lacks the ability for me to create my customized format and it messages up certain MathJax operations. Thus I created a fully customizable parser that converts a markdown or iPynb file to a Overleaf project with assurance that all MathJax and Latex functionality is converted properly and correctly. It also allows for upload of your own customizable Latex formatting file. Apparently, I was not the only person with this level of frustration with NBConvert, shortly after mentioning my personal project with a Data Science professors at Cal Poly, he asked me if I would be open to making the project opened source and for all to use. As of now, the only notable users I know of is that Data Science professor and a Senior Data Scientist at Google. For more about this project, visit the GitHub repo at https://github.com/bfkwong/Latex-Book-Parser
